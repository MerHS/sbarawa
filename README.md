# SBARAWA: Flutter drum score application for drummers by a drummer

## Features

- WYSIWYG Piano roll + Drum score
- Evenly spaced notes & Breaks per quarter notes
- Configurable bars
  - Number of bars per line
  - Adjust width for each bar
- Fully configurable set of notations (globally or locally)
- Export to PDF
